## idをつけよう
spanタグそれぞれにidを付けてjavascriptから引っ張れるようにしましょう。
```html
<!DOCTYPE html>
<html>
    <head>
        <title>文章生成マシーン</title>
    </head>
    <body>
        <h1>文章生成マシーン</h1>
        <button>生成</button>
        <div>
            <span id="text1">あなたは</span>
            <span id="text2">耳が</span>
            <span id="text3">良い</span>
        </div>

        <script src="script/index.js"></script>
    </body>
</html>
```

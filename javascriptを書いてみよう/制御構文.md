## 繰り返し処理
### for
単純にx回繰り返したい場合はforを用います。
forの文法は以下です。
```javascript
for (var 変数 = 初期値; forを継続させる条件; forが1巡した後の処理) {
}
```

例
```javascript
for (var i = 0; i < 10; i++) {
    console.log(i);
}
```
この場合だと、初期値を0、iが10以下ならfor継続、for1巡時
